# MotionSwiper

DEMO: https://motion-swiper.vercel.app/

Codepen: [https://codepen.io/jramke/pen/zYmdeeX?editors=0010](https://codepen.io/jramke/pen/KKGOqZj?editors=1010)

MotionSwiper is a draggable slider with a fluid bending motion effect. This effect is mostly created with threejs.

Something like this: https://codepen.io/ReGGae/pen/povjKxV from https://jesperlandberg.dev/

This swiper does not use threejs. Its realized with SVG Clippaths and [GSAP](https://greensock.com/gsap/) to animate them.


## Getting started

To use the swiper you have to include GSAP.
